# Team JAVA
1. Exception caught: IllegalArgumentException. Can't read the matrix. Имя матрицы не с заглавной буквы.
	Не пропущена строка после объявления матриц.
	Несколько матриц объявлены в одной строке.
	Матрица не однородна.
	При создании матрицы присутствуют символы, кроме пробелов, точек с запятыми, целых чисел .
	Элементы матрицы не заключены в квадратные скобки. Несколько матриц объявлены под одинаковым именем.
	Если нет скобок или есть только одна.
2. Exception caught: IllegalArgumentException. Can't read operations. 
	Матрицы с операндами разделены несколькими строками или не разделены вовсе.
	В операциях с матрицами присутствуют символы, кроме названий матриц, +, -, * .
	В операциях с матрицами есть несколько идущих подряд операций.
3. Exception caught: MatrixOperationException. Can't perform 
operations.
	Неподходящая для умножения размерность . 
	Неподходящая для сложения размерность. 
	Неподходящая для вычитания размерность.

# Team PHP 
Illegal Argument. Can`t read the matrix.
	1. Имя матрицы не с заглавной буквы
	2. Повторяющиеся названия для матриц
	3. Каждая матрица не на отдельной строке
	4. Матрицы не однородны	
	5. Элементы матриц не имеют тип integer
	6. Перед выражением операторов отсутсвует пустая строка
Illegal Argument. Can`t read operations
	1. Невозможно выполнить сложение
	2. Невозможно выполнить вычитание
	3. Невозможно выполнить произведение
Illegal Argument. Can`t read operations
	1. Идущие подряд несколько операндов
	2. Идущие подряд несколько операторов
	3. Присутствие в выражении неразрешённых операторов
	4. Выражение находится не на одной строке
	5. Наличие операторов в начале и конце строки
